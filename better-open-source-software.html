Better Open Source Software

For any type of software, which is Better?

For any specific software, how can it get Better?

Better aims to utilize ingcentive systems to make it profitable to have good ideas and to implement them.

Free Open Source Software (FOSS) is traditionally released out of the goodness of one’s heart, and it is rarely profitable.

Today’s FOSS ecosystem is like that of stagnant water - BOSS aims to be the pump which turns it into a flowing river.
